# Secretar-POCHY
Секретарь ПОЧУ выводит информацию о техникуме.
Можно узнать адрес, специальности, стоимость и контакты техникума.
При нажатии на кнопку "Адрес техникума", выйдет реальный адрес техникума.
При нажатии на кнопку "Специальности", выйдут специальносьти, на которые ты можешь поступить.
При нажатии на кнопку "Стоимость обучения", выйдет примерная стоимость обучения.
При нажатии на кнопку "Связь с нами", выйдут контакты техникума.
Бот написан в Pythone, но предназначен только в Telegram.
Для использования кода нужно скачать Pythone(IDLE), скопировать код, затем вставить код в новый файл(IDLE).
В коде нужно изменить TOKEN на свой TOKEN в боте @BotFather.
Чтобы бот заработал нужно нажать кнопку /start.
